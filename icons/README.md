# Split View Chrome Extension 图标说明

## 需要的图标文件

请在此目录中放置以下尺寸的图标文件：

### 必需图标
- `icon16.png` - 16x16 像素，用于扩展管理页面
- `icon32.png` - 32x32 像素，用于Windows等系统
- `icon48.png` - 48x48 像素，用于扩展管理页面
- `icon128.png` - 128x128 像素，用于Chrome Web Store

## 图标设计建议

### 主题风格
- **色彩方案**：使用蓝色和紫色的渐变，象征分屏和现代化
- **主要元素**：包含两个屏幕或窗口的图标设计
- **简洁明了**：在小尺寸下也能清晰识别

### 设计元素
1. **分屏符号**：两个并排的矩形框
2. **箭头或分割线**：表示分割功能
3. **现代化风格**：扁平化设计，圆角矩形

### 推荐设计
```
┌──────────────┐  ┌──────────────┐
│   Screen 1   │  │   Screen 2   │
│              │  │              │
│              │  │              │
└──────────────┘  └──────────────┘
```

### 临时解决方案
在开发阶段，您可以：
1. 使用Chrome默认图标（不设置icons字段）
2. 使用在线图标生成器创建临时图标
3. 从免费图标库下载合适的图标

### 推荐的在线工具
- **Favicon.io**: https://favicon.io/
- **IconMaker**: https://iconmaker.app/
- **Canva**: https://www.canva.com/

### 颜色参考
- 主色：#2563eb (蓝色)
- 辅色：#7c3aed (紫色)
- 背景：白色或透明

## 文件要求
- 格式：PNG
- 背景：透明或白色
- 质量：清晰，无锯齿
- 大小：符合上述尺寸要求
