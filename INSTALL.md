# 🚀 Split View Chrome Extension 安装使用指南

## 📦 安装方法

### 步骤 1: 下载扩展文件
1. 确保您已下载所有扩展文件到本地文件夹
2. 文件夹应包含以下文件：
   ```
   split-view/
   ├── manifest.json
   ├── popup.html
   ├── popup.css
   ├── popup.js
   ├── background.js
   ├── guide.html
   ├── README.md
   ├── INSTALL.md
   ├── test.js
   └── icons/
       └── README.md
   ```

### 步骤 2: 在Chrome中加载扩展
1. **打开Chrome扩展管理页面**
   - 在地址栏输入：`chrome://extensions/`
   - 或者：点击右上角三点菜单 → 更多工具 → 扩展程序

2. **启用开发者模式**
   - 在扩展管理页面右上角，打开"开发者模式"开关
   - 开启后会显示额外的加载选项

3. **加载扩展**
   - 点击"加载已解压的扩展程序"按钮
   - 在文件选择器中，选择 `split-view` 文件夹
   - 点击"选择文件夹"

4. **确认安装**
   - 如果一切正常，扩展会出现在扩展列表中
   - Chrome工具栏会显示Split View图标（如已添加图标）

## 🎯 使用方法

### 首次使用设置

1. **打开扩展**
   - 点击Chrome工具栏中的Split View图标
   - 扩展会自动检测Chrome Split View功能状态

2. **启用Chrome Split View（如需要）**
   - 如果显示"Split View未启用"，点击"启用指南"
   - 按照指南步骤启用Chrome实验性功能：
     - 打开 `chrome://flags/#split-view`
     - 找到Split View选项
     - 设置为"Enabled"
     - 重启Chrome

### 基本使用

1. **选择网站**
   - **左侧网站**：从下拉菜单选择，或选择"当前页面"
   - **右侧网站**：从预设列表选择，或选择"自定义网址"

2. **选择分屏模式**
   - **左右分屏**：两个窗口并排显示（推荐）
   - **上下分屏**：两个窗口垂直排列
   - **专注模式**：主窗口较大，副窗口较小

3. **开始分屏**
   - 点击"开始分屏浏览"按钮
   - 系统会自动创建两个优化布局的窗口

### 快速模板使用

扩展提供了4个预设模板：

- **💼 工作模式**：GitHub + Google Docs
- **📚 学习模式**：Google + Wikipedia
- **🤖 AI对话**：ChatGPT + Claude
- **🎮 娱乐模式**：Bilibili + Zhihu

点击对应模板按钮即可快速应用。

### 高级设置

1. **记住布局**：保存上次的分屏设置和网站选择
2. **自定义网址**：支持输入任意网址进行分屏

## 🔧 故障排除

### 常见问题

**Q: 扩展无法加载**
```
A: 检查以下几点：
1. 确保manifest.json格式正确
2. 确保所有文件都在同一文件夹中
3. 检查Chrome版本是否支持Manifest V3
```

**Q: 点击图标没有反应**
```
A: 尝试以下解决方案：
1. 重新加载扩展（点击刷新按钮）
2. 重启Chrome浏览器
3. 检查是否有权限限制
```

**Q: Split View功能无法使用**
```
A: 这是因为Chrome原生Split View功能：
1. 可能已在您的Chrome版本中移除
2. 需要启用Chrome实验性功能
3. 可以使用扩展的替代方案（窗口分屏）
```

**Q: 分屏窗口位置不正确**
```
A: 调整方法：
1. 手动拖动窗口到合适位置
2. 检查屏幕分辨率设置
3. 尝试不同的分屏模式
```

**Q: 自定义网址无法打开**
```
A: 检查网址格式：
1. 确保包含协议（http://或https://）
2. 确保网址拼写正确
3. 网站是否可以正常访问
```

### 调试方法

1. **查看错误信息**
   - 右键点击扩展图标 → "检查弹出内容"
   - 查看Console面板的错误信息

2. **检查扩展状态**
   - 在扩展管理页面查看扩展状态
   - 查看是否有错误提示

3. **重新安装**
   - 删除扩展后重新加载
   - 确保文件完整性

## 💡 使用技巧

1. **键盘快捷键**
   - 在分屏模式下使用 `Ctrl+Tab` 在窗口间切换

2. **网站组合推荐**

   - 使用模板快速切换工作场景
